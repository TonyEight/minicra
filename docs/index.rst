MiniCRA - Documentation
=======================

Welcome to the documentation page. **MiniCRA** is a simple Web application 
allowing registered users to manager their *activity reports*.

It is based on :

* `Django <https://www.djangoproject.com>`_
* `Bootstrap <http://getbootstrap.com>`_
* with the flavor of `FontAwesome <http://fontawesome.io>`_
